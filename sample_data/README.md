# Blood Cell Sample Dataset

This folder contains curated demo images for the Blood Cell Detection System.

## Categories

### Healthy Samples
Images containing:
- RBC
- WBC and/or Platelets
- No malaria parasite

### Malaria Samples
Images containing:
- RBC
- Malaria Parasite

### Advanced Samples
Images containing:
- RBC
- Malaria Parasite
- Additional blood components (WBC or Platelets)

## Purpose

This dataset is provided for demonstration and testing of the Blood Cell Detection System web application.

Users can:
- Upload their own image
- Use a random sample
- Select healthy samples
- Select malaria samples
- Select advanced samples

without requiring external datasets.
